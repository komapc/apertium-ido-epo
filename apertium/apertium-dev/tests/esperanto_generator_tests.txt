## ROOT CAUSE

All # errors stem from:
1. Bilingual -o flag strips tags when entry is base form only
2. Transfer receives words without complete tag sets
3. Generator can't produce forms without proper tags

SOLUTIONS ATTEMPTED:
✅ Added missing bilingual entries (havar→havi, linguo→lingvo)
✅ Added transfer tense conversion rules (pri→pres)
⚠️ Added default tense in otherwise clause (not triggering properly)
